Nexon Reloaded — FiveM Resource

NXN
Notifications

Moduláris, konfigurálható in-game értesítési rendszer FiveM RP szerverekhez. Lua-alapú logika, NUI renderelés, teljes export API — más resource-okból azonnal használható.

gamepadFiveM check_circleLua 5.4 integration_instructionsExport API eventEvent-alapú palette5 típus

Telepítés

#install

Másold az nxn-notifications mappát a szerver resources könyvtárába, majd add hozzá a server.cfg-hez.

server.cfg 
# Más resource-ok ELŐTT töltsd be
ensure nxn-notifications
lightbulb Az nxn-notifications-t minden más resource elé érdemes betölteni, hogy az exportok elérhetők legyenek induláskor.

Konfiguráció

#config

A config.lua tartalmaz minden testreszabható értéket. Az alábbi táblázat az összes opciót leírja.

config.lua — teljes referencia 
Config.Stack = {
    maxVisible = 5,      -- Egyszerre látható max notifikáció
    maxQueue   = 20,     -- Queue mérete
    direction  = "down"  -- "down" | "up"
}

Config.Position = {
    stack   = "top-right",    -- "top-right"|"top-left"|"bottom-right"|"bottom-left"
    prompt  = "bottom-center", -- "bottom-center"|"bottom-left"|"bottom-right"
    offsetX = 32,
    offsetY = 96
}

Config.Timing = {
    defaultDuration = 5000,  -- ms (0 = végtelen)
    animateIn       = 350,
    animateOut      = 300
}

Config.Animation = {
    enterStyle = "slide-fade",  -- "slide-fade"|"slide"|"fade"|"scale"
    exitStyle  = "slide-fade",
    errorShake = true
}

Értesítés típusok

#types

Az 5 elérhető típus — kattints bármelyikre az előnézet megtekintéséhez.

Simple
Ikon + kategória + cím. Gyors visszajelzésekhez.
Status
Progress bar + érték. Bio-statisztikákhoz, töltésekhez.
Interaction
Játékos-interakció. Név, ID, üzenet, folyamat.
Discovery
Helyszín felfedezés. Kép + cím + meta infók.
Prompt
Interakció prompt a képernyő alján. Billentyű + szöveg.

Server Exportok

#server-exports

Szerver oldalról (server.lua) hívható exportok — source a játékos szerver ID-ja.

SERVER Send* exportok + Broadcast* expand_more
Minden kliens export-hoz van egy Send prefixű szerver változat. A Broadcast* exportok összes online játékosnak küldenek.
Lua — server.lua 
-- Egy játékosnak
exports['nxn-notifications']:SendSimple(source, {
    title = "Figyelmeztetés",
    icon  = "warning",
    color = "warning",
})

exports['nxn-notifications']:SendStatus(source, {
    title   = "HP: 45%",
    value   = 45,
    icon    = "favorite",
    color   = "error",
})

exports['nxn-notifications']:SendDismiss(source, notifId)
exports['nxn-notifications']:SendDismissAll(source)

-- Összes játékosnak (broadcast)
exports['nxn-notifications']:BroadcastSimple({
    title    = "Szerver újraindul 5 perc múlva",
    icon     = "schedule",
    color    = "warning",
    duration = 10000,
})
Szerver exportLeírás
SendSimple(src, opts)Simple küldése egy játékosnak
SendStatus(src, opts)Status küldése egy játékosnak
SendInteraction(src, opts)Interaction küldése
SendDiscovery(src, opts)Discovery küldése
SendPrompt(src, opts)Prompt küldése
SendDismiss(src, id)Elrejtés egy játékosnál
SendDismissAll(src)Összes elrejtése egy játékosnál
SendUpdate(src, id, opts)Frissítés egy játékosnál
BroadcastSimple(opts)Simple küldése mindenkinek
BroadcastStatus(opts)Status küldése mindenkinek

Esemény-alapú hívás

#events

Ha nem szeretnéd az export szintaxist, TriggerEvent-tel is hívhatod a rendszert. Kliens oldalon belül működik.

EVENT nxn:notify  ·  nxn:dismiss  ·  nxn:dismissAll  ·  nxn:update expand_more
Lua — client esemény hívás 
-- Megjelenítés (type mező kötelező!)
TriggerEvent('nxn:notify', {
    type     = "simple",
    title    = "Hello világ",
    icon     = "waving_hand",
    color    = "success",
})

TriggerEvent('nxn:notify', {
    type  = "status",
    title = "Thirst: 60%",
    value = 60,
    icon  = "water_drop",
})

-- Elrejtés
TriggerEvent('nxn:dismiss', id)
TriggerEvent('nxn:dismissAll')

-- Frissítés
TriggerEvent('nxn:update', id, { value = 30, title = "Thirst: 30%" })

Kódpéldák

#examples

Hunger/Thirst rendszer integráció

Lua — hunger system 
local hungerNotifId = nil

local function UpdateHunger(value)
    if value < 30 then
        if hungerNotifId then
            -- Már látható, csak frissítjük
            exports['nxn-notifications']:Update(hungerNotifId, {
                title = "Hunger: " .. value .. "%",
                value = value,
            })
        else
            -- Első megjelenítés
            hungerNotifId = exports['nxn-notifications']:ShowStatus({
                title    = "Hunger: " .. value .. "%",
                icon     = "fastfood",
                category = "Bio-Stats",
                value    = value,
                subtext  = "Immediate Nutrients Required",
                color    = "error",
                duration = 0,  -- végtelen
            })
        end
    elseif hungerNotifId then
        exports['nxn-notifications']:Dismiss(hungerNotifId)
        hungerNotifId = nil
    end
end

Átmotozás / cuff interakció

Lua — search interaction 
local function StartSearch(targetName, targetId)
    local id = exports['nxn-notifications']:ShowInteraction({
        name     = targetName,
        playerId = targetId,
        message  = "Átkutatja a személyedet...",
        progress = 0,
    })

    local duration = 5000
    local steps    = 20
    local interval = duration / steps

    CreateThread(function()
        for i = 1, steps do
            Wait(interval)
            exports['nxn-notifications']:Update(id, {
                progress = (i / steps) * 100
            })
        end
        exports['nxn-notifications']:Dismiss(id)
    })
end

Bevált minták

#patterns
check_circle Végtelen notifikáció kezelése: mindig tárold el a visszakapott ID-t, és hívj Dismiss(id)-t, ha már nem szükséges. Ne támaszkodj DismissAll()-ra, mert az más resource notifikációit is eltünteti.
warning Queue limit: ha sokat hívsz egymás után, a maxQueue limit felett az újabb notifikációk eldobódnak. Kritikus értesítéseknél adj egyedi id-t és használj Update()-t.
info Prompt + zóna: ShowPrompt-ot Enter zone eseménynél jelenítsd meg, és Exit zone-nál hívj Dismiss(promptId)-t. A promptId-t globálisan tárold a scriptedben.
